<strong>Magento Search configuration settings:</strong>

To configure, go to <em>Magento Admin Panel &gt; Configuration &gt; Catalog &gt; Catalog&gt; Catalog search</em>. Open a “Catalog Search” item and proceed with following settings.<img class="aligncenter wp-image-188" src="https://blog.expertrec.com/wp-content/uploads/2017/07/magento-search-terms1.png" alt="magento search " width="608" height="380" />

1. <b>Tracking search queries using google analytics-</b>You cant fix what you cant measure.The number one thing to set up on day one is to track your website queries using the free tool google analytics. Magento admin gives basic search analytics that is not enough to do detailed analysis, so it is better to set up google site search tracking from day one. This is pretty simple to do- you have to tell google analytics what is the search format of your store with the query parameter. My article on how to set up search tracking in magento stores using google analytics can be found <a href="https://blog.expertrec.com/track-magento-search-queries-using-google-analytics/" target="_blank" rel="noopener">here</a>. Also if you need help in adding google analytics to your magento store read <a href="https://blog.expertrec.com/add-google-analytics-magento-store-ecommerce-tracking/" target="_blank" rel="noopener">this</a> article.<a href="magento search"><img src="https://blog.expertrec.com/wp-content/uploads/2017/07/site_search_1.png" alt="" width="1342" height="553" class="alignnone size-full wp-image-979" /></a>

2. <strong>Minimal Query Length</strong> - Minimum number of characters that you want your users to enter while searching.<img class="aligncenter wp-image-564 size-full" src="https://blog.expertrec.com/wp-content/uploads/2017/09/magento-catalog-search.png" alt="magento search settings" width="700" height="253" />

2. <strong>Maximum Query Length</strong> - Maximum number of characters that you want your users to enter while searching

3. <strong>Set the value of Maximum Query Words Count</strong> - Maximum number of words your users can enter for searching.

4.<strong> Search Type- </strong>Set it to "Full Text"
<ul>
 	<li><strong>Like</strong> - This type will show results even with partial match of search query to product. Includes only search results with the partial matching of search. .</li>
 	<li><strong>Fulltext</strong> - Includes only search results with the exact match to the search query.</li>
 	<li><strong>Combine</strong> - it's a combination of mentioned above search types.</li>
</ul>
5. <strong>Layered Navigation</strong> - Determines the maximum value of facets, which can be displayed for a search query. If you set this to '0' then this is will result in showing all the results. It is not recommended to use 0 if you have a lot of attributes, since it might slow down your magento search.<img class="aligncenter wp-image-156" src="https://blog.expertrec.com/wp-content/uploads/2017/08/layered-navigation1.png" alt="magento search settings" width="662" height="248" />

6. <strong>Magento advanced catalog search - </strong>You will usually find this link at the bottom of the page near the footer section. To know how to enable this, read here- <a href="https://magento.stackexchange.com/questions/42066/how-to-display-the-advanced-search-on-the-front-page">how to display magento advanced search on home page </a>

<img class="aligncenter wp-image-189" src="https://blog.expertrec.com/wp-content/uploads/2017/07/magento-search-terms-2.jpg" alt="magento search settings" width="284" height="353" />

To add more attributes to this, go to Catalog-&gt; Attributes-&gt;Manage Attributes-&gt;edit<img class="alignnone wp-image-190 size-full" src="https://blog.expertrec.com/wp-content/uploads/2017/07/magento-search-terms-3.png" alt="magento search settings" width="1197" height="277" />

7. <strong>Managing magento popular search terms:</strong>

<img class="wp-image-191 aligncenter" src="https://blog.expertrec.com/wp-content/uploads/2017/07/magento-search-settings.jpg" alt="magento search settings" width="715" height="189" />

This is again found in the footer sections. This gives you an idea of what queries people are searchin on your site. This is a word cloud and the more the font of a site, the more searched it is. To manage search terms, go to <em>magento admin panel-&gt;catalog-&gt;search terms-&gt; choose a search term-&gt; edit <img class="wp-image-192 aligncenter" src="https://blog.expertrec.com/wp-content/uploads/2017/07/magento-search-settings-2.png" alt="magento search settings" width="699" height="332" /></em>

<img class="aligncenter wp-image-177" src="https://blog.expertrec.com/wp-content/uploads/2017/08/magento-URL-redirect-2-1024x436-1.png" alt="magento search settings" width="698" height="297" />

Here you will be able to -
<ol>
 	<li dir="ltr">See number of searches.</li>
