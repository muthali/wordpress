Google site search is being discontinued as of April 1st 2018. Google custom search is an easy alternative to switch to but contains ads. I have mentioned a list of Google site search alternatives in this article - <a href="https://blog.expertrec.com/google-site-search-alternative/" target="_blank" rel="noopener">link</a>. Some of these are ready made solutions that can be plugged and used. Expertrec is one of such soltuions. Here are steps that you can follow -

Steps for migration-
<ul>
 	<li>Go to <a href="https://cse.expertrec.com" target="_blank" rel="noopener">cse.expertrec.com</a>.</li>
 	<li>Login with gmail.</li>
 	<li>Enter site details.</li>
 	<li>Get code.</li>
 	<li>Initiate crawl.</li>
 	<li>Check crawl status.</li>
 	<li>Check search demo.</li>
 	<li>Take search live.</li>
 	<li style="text-align: left;">Choose search UI type (optional)</li>
</ul>
<p style="text-align: center;">[inbound_button font_size="20" color="#c8232b" text_color="#ffffff" icon="" url="https://cse.expertrec.com/" width="" target="_blank"]Replace my Google site search[/inbound_button]</p>
Here are detailed instructions-
<ol>
 	<li><strong>Go to <a href="https://cse.expertrec.com" target="_blank" rel="noopener">https://cse.expertrec.com</a></strong> and click on Google site search replacement-<img src="https://blog.expertrec.com/wp-content/uploads/2018/01/1.png" alt="Google site search discontinued" width="1251" height="621" class="aligncenter wp-image-851 size-full" /></li>
 	<li><strong>Login</strong>-Sign in with your Google account-<img src="https://blog.expertrec.com/wp-content/uploads/2018/01/2.png" alt="Google site search discontinued" width="1124" height="583" class="aligncenter wp-image-852 size-full" /></li>
 	<li><strong>Enter site details</strong>-Enter the URL of the website you want to crawl and choose your nearest data center region.<img src="https://blog.expertrec.com/wp-content/uploads/2018/01/3.png" alt="Google site search discontinued" width="1227" height="560" class="aligncenter wp-image-853 size-full" /></li>
 	<li><strong>Add Code</strong>-Now you will get a code to copy. Please add this code to a div element in your site's body section. You need to add this on all pages where you want the search box to appear. Please note that this will also add the search box UI. You could also replace your Google site search code with our code.<img src="https://blog.expertrec.com/wp-content/uploads/2018/01/4.png" alt="Google site search discontinued" width="1157" height="646" class="aligncenter wp-image-854 size-full" /></li>
 	<li><strong>Initiate Crawl </strong>-Crawl/ indexing of your pages will start. <img src="https://blog.expertrec.com/wp-content/uploads/2018/01/5.png" alt="Google site search discontinued" width="1127" height="604" class="aligncenter wp-image-855 size-full" /></li>
 	<li><strong>Check crawl status</strong>- You will get a control panel page to tweak your custom search features. Here you can check your demo. <img src="https://blog.expertrec.com/wp-content/uploads/2018/01/7.png" alt="Google site search discontinued" width="1309" height="490" class="aligncenter wp-image-856 size-full" /></li>
 	<li><strong>Check search demo - </strong>You can check the demo before taking live, by clicking on the demo link<img src="https://blog.expertrec.com/wp-content/uploads/2018/01/10.png" alt="Google site search discontinued" width="1334" height="641" class="aligncenter wp-image-858 size-full" /></li>
 	<li><strong>Take live</strong> - once you add the code to your site, search goes live. Search box looks similar to the following image. When a user presses enter the user interface looks like the above image.<img src="https://blog.expertrec.com/wp-content/uploads/2018/01/11.png" alt="" width="1094" height="657" class="alignnone wp-image-860 size-full" /></li>
 	<li><strong>Search Layout- </strong>The default UI is the overlay UI where the search results pops up in the same page. Some customers want search results on a different page. For search results to come on a new page, go to Configuration-&gt;search layout-&gt;choose "search layout on different page". In the additional info section add the search result path where you want search results to come on enter and the query parameter. After this, you have to get the new code and add it to your site to get the new search interface.<img src="https://blog.expertrec.com/wp-content/uploads/2018/01/searchh-results.jpg" alt="Google site search discontinued" width="1041" height="505" class="alignnone wp-image-899 size-full" />In the below example, search result path is :"/search-results/" and <strong>q</strong> is the "query parameter" if your search results have the URL format <a href="http://providerconnection.davisvision.com/search-results/?q=lense" target="_blank" rel="noopener">http://providerconnection.davisvision.com/search-results/?q=lense</a>                   <img src="https://blog.expertrec.com/wp-content/uploads/2018/01/14.png" alt="Google site search discontinued" width="898" height="841" class="aligncenter wp-image-868 size-full" /></li>
 	<li><strong>Fine tuning Look and feel of the search interface-  </strong>A lot of our customers moving from google site search requested us to add this feature that would help them choose their own colors and font sizes for search results and we have obliged. Go to <strong><em><a href="https://cse.expertrec.com" target="_blank" rel="noopener">cse.expertrec.com</a>-&gt; sign in-&gt;look and feel-&gt; Inputbox</em></strong>. Here you can make adjustments to the search box . You can edit the height, margin, padding, font size etc. You can also add custom css, your own font awesome icon for the search box and much more<img src="https://blog.expertrec.com/wp-content/uploads/2018/01/look-and-feel.jpg" alt="Google site search discontinued" width="1060" height="554" class="alignnone size-full wp-image-901" /></li>
 	<li><strong>Adding multiple URLs to crawl</strong>- In some situations, you might want to crawl more than one url to show in search results. Go to https://cse.expertrec.com-&gt;sign-in-&gt;crawl settings-&gt;Add/edit URL-&gt;enter the URLs you want to crawl -&gt;press update.<img src="https://blog.expertrec.com/wp-content/uploads/2018/01/add-url.jpg" alt="google site search discontinued" width="1336" height="596" class="alignnone size-full wp-image-903" /></li>
 	<li><strong>Enabling Image search</strong>- By default, image search is disabled in expertrec's google site search replacement. To  enable this, go to crawl settings-&gt;custom fields-&gt;enable extractor-&gt;enter the CSS selector and attribute name-&gt;click update<img src="https://blog.expertrec.com/wp-content/uploads/2018/01/css-selector.jpg" alt="google site search discontinued" width="1279" height="555" class="alignnone size-full wp-image-904" /></li>
 	<li><strong>Crawling behind login pages-</strong> Sometimes you might want to crawl pages that are behind a login page. If you enter the following details in our control panel.<img src="https://blog.expertrec.com/wp-content/uploads/2018/01/login.jpg" alt="google site search discontinued" width="767" height="521" class="alignnone size-full wp-image-906" /></li>
 	<li><strong>Filter URLs you don't want to show up in search results- </strong>Go to <em>crawl settings-&gt;URL settings-&gt;Filter URLs-&gt; enter the URls you don't want to come in search results-&gt;Update.<img src="https://blog.expertrec.com/wp-content/uploads/2018/01/url-settings.jpg" alt="Google site search discontinued" width="1039" height="592" class="alignnone size-full wp-image-912" /> </em>Our crawler by default ignores URLs that have GET paramters. To crawl URLs that have GET paramater enable this option<img src="https://blog.expertrec.com/wp-content/uploads/2018/01/get-para.jpg" alt="google site search discontinued" width="1020" height="484" class="alignnone size-full wp-image-913" /></li>
</ol>
If you need any help in migration from your google site search, you can raise a support ticket <a href="https://cloudinfra.freshdesk.com" target="_blank" rel="noopener">here</a>.
